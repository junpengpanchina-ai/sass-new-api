# 管理员用户端：订阅计划

订阅是按周期购买的套餐，购买后在有效期内享受套餐内的配额或特权，适合有稳定用量需求的用户。左侧导航点击「订阅」，或直接访问 `/console/subscription`。

> 说明：当前仓库实现为本地演示（模拟购买成功与订阅状态展示），后续接入 Stripe/EPay 等真实支付与订阅账单。

## 查看订阅套餐

> 截图占位：`docs/assets/subscription-list.png`

订阅套餐列表展示所有可购买的套餐，包含套餐名称、价格、有效期、包含配额等信息。

订阅套餐提供按周期计费的配额包，可选择日、周、月等不同周期的套餐。

## 订阅详情

### 浏览套餐信息

在订阅页浏览可用套餐，查看各套餐的价格、有效期和包含配额。

> 截图占位：`docs/assets/subscription-buy.png`

## 查看当前订阅状态

### 订阅信息

购买订阅后，在订阅页顶部可查看当前套餐的详细信息：

- 套餐名称和有效期截止日期
- 套餐内剩余配额

> 截图占位：`docs/assets/subscription-status.png`

### 自动续费设置

可在此设置到期后是否自动续费（后续接入真实续费扣款逻辑）。
