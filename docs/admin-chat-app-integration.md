# 管理员用户端：聊天应用集成

支持将 New API 的配置导入到多种 AI 聊天应用，方便测试和日常使用。

## 一键导入配置

在控制台 → 令牌页面（`/console/token`）可以对指定令牌进行「导入/配置」。

> 截图占位：`docs/assets/token-import.png`

### 使用方法

1. 在令牌列表页找到要使用的令牌
2. 点击令牌右侧的「导入/配置」按钮
3. 选择目标聊天应用
4. 复制或导入配置信息（Base URL + API Key）

> 说明：当前仓库实现不会保存令牌 Key（安全考虑），导入弹窗会要求你临时粘贴一次 Key 用于复制配置。

## Lobe Chat

Lobe Chat 是一款开源的多模态对话应用，支持多种大语言模型（LLM）和插件扩展，适用于个人助理、知识问答、内容创作等多种场景。

注意：Lobe Chat 目前不支持从 New API 一键导入配置，需要手动填写。

### 配置步骤

1. 打开 Lobe Chat 应用
2. 进入设置页面，找到「语言模型」配置区域
3. 按照以下信息填写配置：
   - **API 地址**：填写 New API 平台的 Base URL
   - **API Key**：填写在 New API 创建的令牌 Key
4. 点击「保存」完成配置
5. 在模型选择中即可看到 New API 提供的所有可用模型

> 截图占位：`docs/assets/lobe-step-1.png`、`docs/assets/lobe-step-2.png`

## 手动配置其他应用

对于不支持一键导入的应用，可以手动配置。

### 配置信息

| 配置项 | 说明 | 获取位置 |
| --- | --- | --- |
| API Base URL | API 接口地址 | 平台首页或令牌页面 |
| API Key | 令牌密钥 | 创建令牌时显示（仅一次） |
| 可用模型 | 支持的模型列表 | 个人设置 → 可用模型 / 模型页 |

### 通用配置步骤

1. 在目标应用中找到「设置」或「配置」入口
2. 找到「自定义 API」或「OpenAI 兼容接口」配置区域
3. 填写 API Base URL 和 API Key
4. 保存配置后即可使用

提示：大多数支持 OpenAI API 的应用都可以通过填写自定义 API 地址的方式接入 New API。
